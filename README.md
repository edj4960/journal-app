# Journal App

This is a journal app built using React, Electron, and a MySQL database. It allows users to create and store journal entries securely on their computer.

## Installation

To use the journal app, follow these steps:

1. Clone the repository to your local machine.
2. Install the necessary dependencies by running npm install.
3. Create .env file and Add MYSQL DB credentials to file (see .env.example)
4. Start the app by running npm start.

## Usage

The journal app allows users to create new journal entries, edit existing entries, and delete entries they no longer need.

All journal entries are stored securely in a MySQL database on the user's computer. This ensures that entries are kept private and can only be accessed by the user.

## Technologies Used

The journal app was built using the following technologies:

* React: A JavaScript library for building user interfaces.
* Electron: A framework for building cross-platform desktop apps using web technologies.
* MySQL: An open-source relational database management system.
